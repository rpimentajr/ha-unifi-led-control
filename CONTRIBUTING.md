# Contributing

Thank you for your interest in contributing.

## Before submitting

Please:

- Search existing Issues.
- Open a Discussion if you have questions.
- Test your changes before opening a Pull Request.

## Pull Requests

Please include:

- Description of the change
- Tested hardware
- Home Assistant version
- UniFi Network version
- Screenshots (if applicable)

## Coding Style

- Use clear YAML formatting.
- Comment complex logic.
- Keep compatibility with current Home Assistant releases.

## Documentation

Documentation improvements are always welcome.

Even small typo fixes are appreciated.
